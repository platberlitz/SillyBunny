What is Pathfinder?

Pathfinder gives the AI intelligent access to your lorebooks. Instead of relying on keywords, it uses AI to predict which lore entries are relevant to the current scene.

Two ways to use Pathfinder:

  1. Tool Mode: The AI can actively browse and search your lorebook using tools (requires tool-calling API)
  2. Pipeline Mode: Automatically predicts and injects relevant lore before each message (works with any API)
